abstract: 'We examine a structural gap in LLM agent memory systems: agents that store experiences and retrieve them by similarity still fail to recall relevant knowledge at the moment of need. We argue that recall — knowing when to consult memory — is fundamentally distinct from retrieval — finding relevant documents given a query. We propose a six-mode failure taxonomy covering the ways recall breaks down, position ambient context loading as an ecological design pattern, and suggest five evaluation metrics for recall effectiveness. The analysis draws on operational experience with a multi-agent system of twelve specialized agents running persistent sessions with episodic memory.'
